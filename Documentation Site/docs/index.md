# Introduction to Winsimple!
## What is it?
Winsimple is a statically included, header only graphics library. 

By using winsimple, you can create a program that uses:


* GUI
* Sound
* Graphics
* Animations
* Clipboard Interaction
* Drag and Drop Operations
