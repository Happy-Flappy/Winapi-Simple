## Classes

### Timer
High-precision timer for measuring elapsed time.

**Public Methods:**
- `Timer()` - Constructor
- `double restart()` - Reset timer and return elapsed seconds
- `double getSeconds() const` - Get elapsed time in seconds
- `double getMilliSeconds() const` - Get elapsed time in milliseconds
- `double getMicroSeconds() const` - Get elapsed time in microseconds

### Vector2f
Simple 2D vector structure.

**Public Variables:**
- `float x` - X coordinate
- `float y` - Y coordinate

### View
Manages viewport and world coordinate transformations.

**Public Methods:**
- `void setRect(RECT viewRect)` - Set world rectangle
- `void setSize(POINT size)` - Set world size
- `void setPos(POINT pos)` - Set world position
- `void setPortRect(RECT portRect)` - Set viewport rectangle
- `void setPortSize(POINT size)` - Set viewport size
- `void setPortPos(POINT pos)` - Set viewport position
- `RECT getRect()` - Get world rectangle
- `POINT getSize()` - Get world size
- `POINT getPos()` - Get world position
- `RECT getPortRect()` - Get viewport rectangle
- `POINT getPortSize()` - Get viewport size
- `POINT getPortPos()` - Get viewport position
- `void zoom(float factor)` - Zoom view by factor
- `void move(POINT delta)` - Move world position
- `void movePort(POINT delta)` - Move viewport position
- `POINT toWorld(POINT pos)` - Convert window coordinates to world coordinates
- `POINT toWindow(POINT pos)` - Convert world coordinates to window coordinates

### Texture
Handles bitmap loading and management.

**Public Variables:**
- `HBITMAP bitmap` - Handle to bitmap
- `int width` - Bitmap width
- `int height` - Bitmap height
- `COLORREF transparencyColor` - Transparency color key (default: CLR_INVALID)

**Public Methods:**
- `Texture()` - Default constructor
- `Texture(std::string path)` - Constructor with file path
- `bool load(std::string path)` - Load bitmap from file
- `bool isValid() const` - Check if texture is valid
- `void setTransparentMask(COLORREF color)` - Set transparency color

### Drawable (Abstract Base Class)
Base class for all drawable objects.

**Public Variables:**
- `int x, y, z` - Position coordinates
- `int width, height` - Dimensions
- `Vector2f scale` - Scale factors

**Public Methods:**
- `virtual void draw(HDC hdc, View &view) = 0` - Pure virtual draw method
- `virtual bool contains(POINT pos) = 0` - Pure virtual point containment check
- `void setScale(Vector2f s)` - Set scale factors
- `void setOrigin(POINT pos)` - Set origin point
- `int getScaledWidth() const` - Get scaled width
- `int getScaledHeight() const` - Get scaled height
- `POINT getScaledOrigin()` - Get scaled origin

### Sprite (Inherits from Drawable)
Drawable sprite with texture support.

**Public Methods:**
- `Sprite()` - Constructor
- `void setTexture(Texture &texture)` - Set texture reference
- `void setTextureRect(RECT r)` - Set texture sub-rectangle
- `RECT getTextureRect()` - Get current texture rectangle
- `Texture &getTexture()` - Get texture reference
- `const Texture* getTexture() const` - Get const texture pointer
- `bool hasTexture() const` - Check if sprite has texture

### Shape (Inherits from Drawable)
Simple colored rectangle shape.

**Public Variables:**
- `COLORREF color` - Fill color (default: RGB(125,255,255))

**Public Methods:**
- `Shape()` - Constructor

### Line (Inherits from Drawable)
Drawable line segment.

**Public Variables:**
- `POINT start` - Start point
- `POINT end` - End point
- `COLORREF color` - Line color (default: RGB(0,0,255))
- `int width` - Line width (default: 2)

**Public Methods:**
- `Line(POINT start, POINT end, int width, COLORREF color)` - Constructor
- `bool intersects(Line &otherLine)` - Check if lines intersect

### Window
Main window class for creating and managing windows.

**Public Variables:**
- `HWND hwnd` - Window handle
- `int x, y, width, height` - Window position and dimensions
- `bool isTransparent` - Transparency status
- `View view` - Current view

**Public Methods:**
- `Window(int width, int height, std::string title, DWORD style)` - Constructor
- `void setView(View &v)` - Set current view
- `void setChromaKey(COLORREF color, BYTE alpha, std::string type)` - Set window transparency
- `void setLayerAfter(HWND lastHwnd)` - Set window Z-order
- `void setFullscreen(bool fullscreen)` - Toggle fullscreen mode
- `void toggleFullscreen()` - Toggle fullscreen
- `bool getFullscreen() const` - Get fullscreen status
- `bool isOpen()` - Check if window is open
- `bool pollEvent(MSG &message)` - Poll window events
- `void clear(COLORREF color)` - Clear window with color
- `void draw(Drawable &draw)` - Draw a drawable object
- `void display()` - Display rendered content

## Utility Functions

### String Conversion
- `std::wstring WIDE(std::string str)` - Convert UTF-8 string to wide string
- `LPCSTR TO_LPCSTR(std::string &str)` - Convert std::string to LPCSTR

## Global Namespace Functions

### Input Functions (ws::Global)
- `POINT getMousePos(Window &window)` - Get mouse position in world coordinates
- `POINT getMousePos()` - Get mouse position in screen coordinates
- `bool getMouseButton(int vmButton)` - Check mouse button state
- `bool getKey(char vmKey)` - Check keyboard key state

## Usage Notes

1. **Window Creation**: Create a Window object with desired dimensions and title
2. **Drawing Loop**: Use `isOpen()` in a loop, call `clear()`, `draw()`, and `display()`
3. **Event Handling**: Use `pollEvent()` to process window messages
4. **View Management**: Use View object for coordinate transformations and zooming
5. **Transparency**: Use `setChromaKey()` for transparent windows with color keying

## Dependencies
- Windows API (windows.h)
- Standard C++ libraries (iostream, string, map, queue, etc.)
